Portafolio Personal
Descripción
Este proyecto es un portafolio personal diseñado para mostrar mis habilidades, proyectos y experiencia. El portafolio está construido utilizando tecnologías modernas y sigue las mejores prácticas de diseño y desarrollo web.

Tecnologías Utilizadas
- HTML5: Estructura del contenido.
- CSS3: Estilos y diseño responsivo.
- JavaScript: Interactividad y funcionalidad.
- GitHub Pages: Hosting del portafolio.

## Características
- Diseño Responsivo: Adaptable a diferentes tamaños de pantalla (móviles, tabletas y desktop).
- Navbar Interactiva: Barra de navegación con enlaces a diferentes secciones del portafolio.
- Sección de Proyectos: Muestra los proyectos destacados con descripciones y enlaces.
- Sección de Contacto: Formulario de contacto para que los visitantes puedan comunicarse conmigo.
